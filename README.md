# ML-OnlineRetail

## Cara Install

`$ > git clone https://github.com/xazhurea/ML-OnlineRetail.git` 

`$ > cd ML-OnlineRetail`

`$ > python -m venv .venv`

`$ > pip install streamlit`

Test that the installation worked

`$ > streamlit hello`

`$ > pip install joblib`

`$ > pip install scikit-learn`

Running program
`$ > streamlit run app.py`
